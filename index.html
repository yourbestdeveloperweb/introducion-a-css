Introducción a CSS

1- Definición de CSS

Es un lenguaje de estilo utilizado para describir la presentación de un documento HTML o XML. Permite separar el contenido de un documento de su diseño visual, facilitando así la gestión y mantenimiento del diseño de una página web.

2- Funcionamiento Básico de CSS

CSS funciona aplicando reglas de estilo a los elementos del documento. Cada regla especifica cómo deben mostrarse los elementos seleccionados, permitiendo el control del diseño, la tipografía, los colores, el espaciado y otros aspectos visuales.

3- Incluir CSS en un archivo HTML

CSS puede incluirse en un archivo HTML de tres maneras:

  • Inline: Directamente en el elemento HTML utilizando el atributo style. 
    <p style="color: blue;">Texto azul</p>
  • Interno: Dentro de un bloque <style> en la sección <head> del HTML. 
    <head>
                <style>
            
             </style>
                </head>
  • Externo: En un archivo CSS separado, enlazado desde el HTML con <link>. 
    <head>
    <link rel="stylesheet" href="styles.css">
</head>

4- Definición de una regla CSS

Una regla CSS consiste en un selector y un bloque de declaración que especifica cómo se deben aplicar los estilos a los elementos seleccionados.

5- Elemento de una regla CSS

  • Regla: Conjunto completo que incluye un selector y un bloque de declaración.
  • Selector: Parte de la regla que especifica a qué elementos se aplicará el estilo (ej. p, .clase, #id).
  • Bloque de declaración: Contiene una o más declaraciones de estilo encerradas entre llaves {}.
  • Propiedad: Aspecto específico del estilo que se va a aplicar (ej. color, font-size).
  • Valor: El valor asignado a la propiedad (ej. red, 16px).

6- Medios CSS

Los medios CSS se utilizan para aplicar estilos específicos según las características del dispositivo donde se visualiza la página. Esto se logra mediante consultas de medios (media queries).

@media (max-width: 600px)
  
    
7- Comentarios CSS

    
Los comentarios en CSS se utilizan para hacer anotaciones en el código y no se muestran en la página. Se encierran entre /* y */.

    
/* Este es un comentario */
p {
    color: blue; /* Comentario al lado de una propiedad */
}


    
8- Sintaxis de la definición de las propiedades CSS

    
La sintaxis para definir propiedades en CSS es la siguiente:

    
selector {
    propiedad: valor;
}

    
Ejemplo:

    
h1 {
    color: green;
    font-size: 24px;
}
